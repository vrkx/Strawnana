# Strawnana
Strawnana is a PWA for hidden apps built in iOS, this allow to unlock or open some of the hidden apps of iOS by using Shortcuts.

- iOS 13-18
- [RunShort](http://icloud.com/shortcuts/64ed8d055c974221aab9410d99550016)
